Seedha-Sauda

Backend API

Peer to Peer file sharing without any hastle and login

API Routes

Route Method Description
/api/session/start POST Start a new session
/api/session/time/:sessionId GET Get the creation time of a session by session ID
/upload POST Upload a single file (for file sharing)
/api/text/upload POST Upload text to a session
/api/text/:id GET Get text content by session ID
/api/file/:id GET Retrieve file by ID (could be based on session ID or filename)
